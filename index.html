02

Users
& Permissions

👑
Internal · Full Access
Owner / Director
Runs the showroom. Full visibility across all pulls, clients, and coverage outcomes. Reviews business performance and relationship value over time.
All Records Reports Coverage Data Permissions
🧑‍💼
Internal · Operational
Showroom Rep
Day-to-day operations. Books appointments, manages pulls, communicates with clients, approves and closes records.
Contacts Appointments Pulls Inventory
🎓
Internal · Limited
Intern ×2
Supporting tasks only. Pull prep, return logging, reminder sending. No client contact info visible. Cannot approve or close pulls.
Pull Prep Returns Reminders
🎨
External · Touchpoint Only
Client — Stylist / Editor / Art Director
Reaches out via email, DM, or phone. Does not log into the CRM. All intake is handled manually by the rep on their behalf. Their journey is tracked, not their access.
03

Pipeline
Stages

Appointment Track
Stage 01
Appointment Requested
Client reaches out. Rep creates record manually in CRM.
Stage 02
Appointment Confirmed
Date, time, rep assigned. Confirmation sent. Calendar event created.
Stage 03
Visited
Appointment completed. Pull finalized during the showroom visit.
Pull begins
Pull Track
Stage 04
Out
Items have left the showroom. Return date set. Calendar event created and Event ID stored.
Stage 05 · Automated
Due Soon
Return date within threshold. Reminder fires to client and rep automatically.
Stage 06 · Automated
Overdue
Past return date with no return logged. Escalation sent to rep and owner.
Items return
Close Track
Stage 07
Returned
Items received back. Condition checked. Inventory status reset to available.
Stage 08
Closed
Coverage logged — event name, platform, credit given. Client reliability score updated.

06

n8n
Automations

Trigger

Appointment confirmed in CRM

⚡
Actions
  • Send confirmation email to client
  • Create Google Calendar appointment event
  • Schedule day-before reminder to client
Trigger

Morning of appointment day

⚡
Actions
  • Send rep brief: client name, pull history, preferences
Trigger

Pull stage set to Out

⚡
Actions
  • Create Google Calendar return event
  • Write Calendar Event ID back to Pull record
  • Set all linked inventory items to: Out
Trigger

Return date updated on Pull record

⚡
Actions
  • Patch existing Calendar event using stored Event ID
Trigger

Daily check — return date within 2 days

⚡
Actions
  • Move Pull stage → Due Soon
  • Send return reminder to client
  • Notify rep
Trigger

Daily check — return date passed, no return logged

⚡
Actions
  • Move Pull stage → Overdue
  • Send escalation to rep and owner
Trigger

Pull stage set to Returned

⚡
Actions
  • Reset linked inventory items to: Available
  • Remove Google Calendar return event
